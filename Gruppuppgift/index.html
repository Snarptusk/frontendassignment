Färger har funnits i HTML sedan väldigt tidigt, och när CSS släpptes i december 1996 fanns det inbakat i språket. Planen med språket var delvis för att ersätta användingen utav färger i HTML-koden. Anledningen till varför CSS användes var för att man ville separera koden från stylingen, för att på så sätt få en renare kod. Separationen ger även mer flexibilitet och kontroll till specifikationerna för presentationsegenskaperna på webbsidan.

Webfärger är färger som visas på hemsidor på internet, och kan beskrivas/specifieras via tre olika sätt i webbläsare. På den här hemsidan kan man läsa om dessa typer på varsin sida som kan nås från menyn högst upp. RGB-triplet, Hexadecimal format och RGBA.
